## 표현식과 문의 개념

### 표현식 

- 하나의 값으로 평가될 수 있는 코드를 의미.

    값으로 평가될 수 있는 문은 모두 표현식


- 리터럴 표현식

    리터럴 값 자체가 표현식으로 사용
    ```
    10          // 숫자 리터럴
    'Hello'     // 문자열 리터럴
    true        // 불리언 리터럴
    ```

- 변수 표현식

    변수는 할당된 값으로 평가
    ```
    let x = 5;
    x  // 변수 표현식으로 평가되면 5가 반환됨
    ```

- 산술 표현식

    산술 연산을 수행하며, 결과 값으로 평가

    ```
    2 + 3      // 5
    10 * 2     // 20
    ```

- 논리 표현식

    논리 연산을 수행하며, 불리언 값으로 평가

    ```
    true && false  // false
    10 > 5         // true
    ```


### 문 (statement)

문(Statement)

프로그램의 실행 흐름을 제어하는 완결된 구문.

자바스크립트에서 명령을 내리는 기본 단위.

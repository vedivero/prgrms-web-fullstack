
## Choice


### if-else

- 기본 구조
    
    조건이 참일 때와 거짓일 때 각각 다른 코드를 실행.

    ```
    if (조건) {
        // 조건이 참일 때 실행
    } else {
        // 조건이 거짓일 때 실행
    }
    ```

### switch

- 기본 구조
    
    조건이 참일 때와 거짓일 때 각각 다른 코드를 실행.

    ```
    switch (표현식) {
        case 값1:
            // 표현식이 값1과 같을 때 실행할 코드
            break;
        case 값2:
            // 표현식이 값2와 같을 때 실행할 코드
            break;
        default:
            // 위 조건이 모두 거짓일 때 실행할 코드
    }
    ```

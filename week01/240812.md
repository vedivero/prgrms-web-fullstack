# 프로젝트란?
- 하나의 프로그램(시스템)을 만들기 위한 일련의 프로세스
- 소스 코드 구현뿐만 아니라, 프로그램 제작을 위한 모든 과정 포함
  - 기획
  - 설계
  - 테스트
  - 배포

---

<br>

# 협업을 잘하려면
- 개발자는 코드로 소통해야 함
- 필수 요소:
  - 코드 공유
  - 문서 공유
  - 진행 철도
  - 담당자

결국엔, 공유!

# 버전 관리 시스템의 종류

버전 관리 시스템은 크게 세 가지로 분류됨:

1. **로컬 VCS**
2. **중앙집중식 VCS**
   - 대표적인 시스템: SVN, CVS 등
3. **분산 VCS**
   - 대표적인 시스템: Git, Mercurial, Bazaar 등
